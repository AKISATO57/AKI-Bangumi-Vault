# Contributing

欢迎 Fork、修改和提交 Pull Request。

## 开发环境

建议使用 Node.js LTS。

```bash
npm install
npm start
```

检查主进程脚本：

```bash
npm run check
```

## 提交建议

- 不要提交 `node_modules/`、`dist/`、`release/`。
- 不要提交真实的 `资料库/收藏数据.json`、封面、备份或 Access Token。
- 修改 `.cmd` 文件时尽量保持 ASCII-only，避免 Windows 终端编码问题。
- 新功能优先保证桌面版可用，再考虑在线版兼容。

## 许可

提交代码即表示你同意你的贡献按本项目的 MIT License 发布。
