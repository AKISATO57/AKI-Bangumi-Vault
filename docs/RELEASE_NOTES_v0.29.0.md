# Bangumi 保管库 v0.29.0

这一版先响应 Bangumi 小组里关于「备份 tag」和「根据 tag 跳转」的建议。

## 新增

- 区分「我的标签」与「公共标签」。
- 新增「补全公共标签」按钮：对当前筛选结果逐条读取 Bangumi 条目详情里的公共标签。
- 左侧标签筛选会显示「我的」和「公共」来源。
- 条目详情页会分开显示我的标签和公共标签。
- 点击标签可在本地筛选同标签条目。
- 公共标签旁的 ↗ 可打开 Bangumi 对应标签页。
- 导出 JSON、CSV、XLSX、DOCX、离线 HTML 和完整 ZIP 时保留标签信息。

## 说明

公共标签不是每次收藏同步都强制补全，因为逐条读取条目详情会增加很多 API 请求。建议先用状态、分类或搜索缩小范围，再点击「补全公共标签」。

## 兼容性

旧版备份中的 `tags` 会继续作为「我的标签」读取；新版会额外保存 `myTags` 和 `publicTags`。
