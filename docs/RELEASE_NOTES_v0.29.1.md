# Bangumi 保管库 v0.29.1 预览版

这是基于 v0.29.0 的时间字段与公共标签修正版。

## 变化

- 公共标签按钮从“补全公共标签”改为“刷新公共标签”。
- 刷新公共标签时会对当前筛选结果重新读取条目公开标签，而不是只处理没有标签的条目。
- 详情页新增/细分时间信息：
  - 收藏时间：优先使用接口返回的收藏创建时间；没有时提示接口未提供，并显示本地首次记录时间。
  - 收藏记录更新：Bangumi 收藏记录的最后更新时间。
  - 吐槽时间：吐槽内容变化时记录；历史数据会用收藏记录更新时间估算。
  - 首次备份：本工具第一次在本地记录该条目的时间。
  - 本地同步：最近一次同步确认该条目的时间。
- 导出数据中增加收藏时间、吐槽时间、本地同步时间。

## 说明

旧数据无法还原真实的历史收藏时间；如果 Bangumi 接口没有提供收藏创建时间，本工具只能保留“本地首次记录时间”。
