# Bangumi 保管库 v0.29.2 预览版

本版继续调整公共标签与时间字段显示。

## 变化

- 公共标签仍完整保存在备份中，但详情页默认只显示前 24 个高频标签，避免看起来比 Bangumi 页面默认展示更多。
- 去掉“接口未提供”的收藏时间占位，不再显示无法确认的收藏时间。
- 去掉“吐槽时间（历史估算）”，避免给出不准确时间。
- 资料库信息中保留 Bangumi 记录更新时间、首次备份时间、本地同步时间。
- 备份历史新增详细变更列表：评分、状态、进度、吐槽、我的标签、公共标签等字段变化会显示 from/to。

## 说明

Bangumi v0 API 当前没有可靠提供第一次收藏时间和吐槽发布时间；本工具不会伪造这些时间，只记录本地备份过程能准确确认的时间。
